Sales Data Analysis using Power BI

* Project Overview

This project focuses on analyzing sales performance using Power BI.
The goal is to transform raw sales data into meaningful business insights through data cleaning, visualization, and dashboard creation.

This project is designed for beginners to understand the complete Power BI workflow.

* Dataset

The dataset is a CSV file containing sales transaction details such as:

Order ID

Order Date

Product

Category

Region

Sales

Quantity

Profit

The dataset was imported directly into Power BI for analysis.
